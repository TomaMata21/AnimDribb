## Getting Started
1. **Prerequisites**: Ensure you have Flutter installed on your system.
2. **Clone this Repository**:
 ```bash
git clone https://github.com/TomaMata21/AnimDribb.git
```
3. **Navigate to the Directory**:
  ```bash
cd AnimDribb
```  
4. **Install dependencies**:
  ```bash
 flutter pub get
```
5. **Run the App**:
 ```bash
 flutter run
```
